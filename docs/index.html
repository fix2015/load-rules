Getting Started

Overview

load-rules aggregates coding rules from multiple GitHub sources into a searchable registry. Install any rules with one command — no cloning repos or copying files.

Quick Start

# Install a rules
npx load-rules install <name>

# Install for Cursor
npx load-rules install <name> --tool cursor

# Install globally
npx load-rules install <name> --global

# Search
npx load-rules search react

# List all
npx load-rules list

Commands

Install

load-rules install <name>                    # Install for Claude Code (default)
load-rules install <name> --tool cursor       # Install for Cursor
load-rules install <name> --tool codex        # Install for Codex
load-rules install <name> --global             # Install globally
load-rules install <name> -o ./path           # Custom output path

Search

load-rules search react                       # Search by keyword
load-rules search "api design" --tag backend  # Search with tag filter
load-rules search react --json                # JSON output

List

load-rules list                                # List all
load-rules list --tag testing                  # Filter by tag
load-rules list --tool cursor                  # Filter by tool
load-rules list --json                         # JSON output

Info

load-rules info <name>                         # Detailed info
load-rules info <name> --json                 # JSON output

Tags & Sources

load-rules tags                                # Show all tags with counts
load-rules sources                             # Show all sources

Sources

Skills are aggregated from multiple GitHub repositories maintained by the community.

Update Registry

load-rules update                              # Fetch latest from GitHub
GITHUB_TOKEN=ghp_xxx load-rules update         # With auth for higher rate limits

JSON Output

load-rules list --json
load-rules search react --json
load-rules info <name> --json

All commands support --json for machine-readable output.

Programmatic API

const { findSkill, searchSkills, installSkill } = require('load-rules');

const results = searchSkills('react', { tag: 'frontend' });
const skill = findSkill('react-expert');
await installSkill(skill, { tool: 'claude-code', global: true });
